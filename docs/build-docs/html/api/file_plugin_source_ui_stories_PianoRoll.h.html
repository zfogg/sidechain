class PianoRoll : public juce::Component, public juce::Timer
#include <PianoRoll.h>

PianoRoll visualizes MIDI data in a piano roll format (7.5.5.1.1).

Features:

  • Vertical piano keys (configurable range)

  • Horizontal timeline synced with audio playback

  • Note rectangles with velocity-based coloring

  • Playback position indicator

  • Animated note highlighting during playback

MIDI data format expected (from MIDICapture::getMIDIDataAsJSON): { “events”: [ {“time”: 0.0, “type”: “note_on”, “note”: 60, “velocity”: 100, “channel”: 0}, {“time”: 0.5, “type”: “note_off”, “note”: 60, “channel”: 0} ], “total_time”: 30.5, “tempo”: 128 }
