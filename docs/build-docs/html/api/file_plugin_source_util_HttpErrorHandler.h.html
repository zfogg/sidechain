class HttpErrorHandler
#include <HttpErrorHandler.h>

HttpErrorHandler - Centralized HTTP error tracking and debug popup system.

In debug builds, shows a popup for every HTTP error so developers can immediately see what’s failing. In release builds, errors are just logged.
