template<>
class Outcome<void>

Outcome<void> specialization for operations that don’t return a value.

Usage: Outcome<void> deleteUser(const juce::String& id) { if (id.isEmpty()) return Outcome<void>::error(“User ID is empty”); // … delete logic return Outcome<void>::ok(); }
