namespace TextEditorStyler

TextEditorStyler - Consistent TextEditor configuration and styling.

Centralizes the repeated styling operations applied to TextEditors throughout the UI (colors, fonts, placeholders, input restrictions).

Usage: TextEditorStyler::style(emailEditor, “Email address”); TextEditorStyler::stylePassword(passwordEditor, “Password”); TextEditorStyler::styleMultiline(bioEditor, “Tell us about yourself…”);
