void drawCircularAvatar(juce::Graphics &g, juce::Rectangle<int> bounds, const juce::Image &image, const juce::String &initials, juce::Colour backgroundColor = juce::Colour(0xff3a3a3e), juce::Colour textColor = juce::Colours::white, float fontSize = 0.0f)

Draw a circular avatar with fallback to initials.

If image is invalid, draws a colored circle with initials.

Parameters:

  • g – Graphics context

  • bounds – Rectangle to draw in

  • image – The avatar image (can be null/invalid)

  • initials – Fallback initials (e.g., “JD” for John Doe)

  • backgroundColor – Background color for initials fallback

  • textColor – Text color for initials

  • fontSize – Font size for initials (0 = auto-calculate from bounds)

juce::String getInitials(const juce::String &displayName, int maxChars = 2)

Generate initials from a display name.

“John Doe” -> “JD”, “alice” -> “A”, “” -> “?”
