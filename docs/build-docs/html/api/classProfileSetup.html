class ProfileSetup : public juce::Component

ProfileSetup - First-time user profile setup wizard.

Features:

  • Profile picture selection/upload

  • Username and display name input

  • Bio input (optional)

  • Skip option for quick start

  • Image preview while uploading

  • Validation and error handling

This component is shown after initial authentication to help users complete their profile before accessing the main feed.
