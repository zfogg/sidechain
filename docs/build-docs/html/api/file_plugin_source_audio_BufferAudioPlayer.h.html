class BufferAudioPlayer
#include <BufferAudioPlayer.h>

BufferAudioPlayer - Handles audio playback from in-memory AudioBuffer.

This class is designed for playing audio directly from juce::AudioBuffer<float> without needing to encode/decode or save to files. Perfect for previewing recorded audio before upload.

Key features:

  • Direct buffer-based playback (no file I/O)

  • Transport controls (play, pause, seek)

  • Playback progress tracking

  • Volume control

  • Automatic resampling to match DAW sample rate
