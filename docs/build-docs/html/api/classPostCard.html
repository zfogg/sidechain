class PostCard : public juce::Component, public juce::Timer

PostCard displays a single post in the feed.

Features:

  • User avatar with circular clip and fallback to initials

  • Username and relative timestamp

  • Waveform visualization with play progress overlay

  • Play/pause button

  • BPM and key badges

  • Like button with count

  • Comment count indicator

  • Share button

The component uses a callback-based API for actions to keep it decoupled from network/audio code.
