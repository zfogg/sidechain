Namespace StringFormatter

Namespace StringFormatter - Number, duration, and display string formatting utilities.

Contents

  • Detailed Description

  • Functions

Detailed Description

Centralizes formatting logic for counts, durations, percentages, and music-specific values like BPM.

Usage: auto countStr = StringFormatter::formatCount(1234); // “1.2K” auto duration = :ref:`exhale_function_namespaceStringFormatter_1a353b37ab79937d3d9c1879fc24a3c73f`(90.5); // “1:30” auto percent = :ref:`exhale_function_namespaceStringFormatter_1a11e64978a7545c55105d27b24e611b1d`(0.75); // “75%”
