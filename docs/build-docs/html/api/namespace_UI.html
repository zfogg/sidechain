Namespace UI

UIHelpers - Common Namespace UI drawing utilities.

Contents

  • Detailed Description

  • Functions

Detailed Description

Provides consistent drawing patterns for cards, badges, buttons, and text to reduce duplication across components.

Usage: // Draw a card with background and border UI::drawCard(g, bounds, Colors::backgroundLight, Colors::border);
