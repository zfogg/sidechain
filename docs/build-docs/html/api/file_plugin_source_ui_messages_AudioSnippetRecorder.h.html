class AudioSnippetRecorder : public juce::Component, public juce::Timer
#include <AudioSnippetRecorder.h>

AudioSnippetRecorder provides UI for recording short audio snippets (max 30 seconds) to send in messages.

Features:

  • Record button (toggle or hold-to-record)

  • Max duration: 30 seconds

  • Timer display during recording

  • Waveform visualization while recording

  • Cancel button to discard recording

  • Preview and send functionality
