namespace StringFormatter

StringFormatter - Number, duration, and display string formatting utilities.

Centralizes formatting logic for counts, durations, percentages, and music-specific values like BPM.

Usage: auto countStr = StringFormatter::formatCount(1234); // “1.2K” auto duration = StringFormatter::formatDuration(90.5); // “1:30” auto percent = StringFormatter::formatPercentage(0.75); // “75%”
