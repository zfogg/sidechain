class HttpAudioPlayer : public juce::ChangeListener
#include <HttpAudioPlayer.h>

HttpAudioPlayer - Handles HTTP/URL-based audio playback for the feed.

In a VST plugin context, we can’t use standard audio output devices. Instead, this class:

  1. Downloads audio from URLs to memory

  2. Decodes using JUCE’s AudioFormatReader

  3. Mixes playback into the plugin’s processBlock output

Key features:

  • URL-based audio streaming

  • Transport controls (play, pause, seek)

  • Playback progress tracking

  • Volume control

  • Audio caching (LRU with memory limit)
