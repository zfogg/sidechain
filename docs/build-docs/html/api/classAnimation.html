class Animation : private juce::Timer

Animation - Timer-based animation with easing functions.

Provides a simple way to animate values over time with various easing functions. The animation runs on the message thread via juce::Timer.

Usage: Animation anim(300, Animation::Easing::EaseOutCubic); anim.onUpdate = [this](float progress) { alpha = progress; repaint(); }; anim.onComplete = [this]() { isAnimating = false; }; anim.start();
